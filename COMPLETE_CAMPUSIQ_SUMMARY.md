### 🚀 How to Use:

**Login:**
1. Open the application
2. Click role tab (Student/Faculty/Admin)
3. Click quick-fill button for demo credentials
4. Click "Sign In"

**Admin - Import Users:**
1. Login as admin
2. Go to "User Management"
3. Click "Template" to download Excel template
4. Fill template with user data
5. Click "Import" and upload file
6. View import summary

**Faculty - Mark Attendance:**
1. Login as faculty
2. Go to "Mark Attendance"
3. Select subject and session
4. Mark P/A/L for each student
5. Click "Submit Attendance"
6. Click "Export to Excel" to download

**Student - View CGPA:**
1. Login as student
2. Go to "CGPA Planner"
3. Enter target CGPA and credits
4. Click "Calculate Required SGPA"
5. View semester-wise grades
